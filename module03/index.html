Let's Explore GIT and Commit

If you want to use git and github you have to follow some steps.

Step to Follow :

  1. Install "git scm" and "node" in your windows
  2. Login / SignUp to your github Account
  3. Open a New repository
  4. Connect the folder to your github repo
  5. Commit and push your code
